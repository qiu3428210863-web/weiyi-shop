# 订货平台项目官网 — 设计文档

## 理解总结

- **做什么**：一个独立的单页项目官网，展示订货平台的完整设计理念
- **为什么做**：作为作品集/招聘展示，体现产品思维、UCD 方法论和设计能力
- **给谁看**：招聘官、面试官、技术博客读者
- **风格**：专业 B2B 产品感，白底、清晰信息层级（Stripe/Linear 风格）
- **技术**：单页 HTML（无构建步骤）
- **内容**：精选 4 个核心模块深度展开，其他概要介绍

## 设计基调

| 用途 | 色值 |
|------|------|
| 主色（强调/标题） | `#1b1c1c`（近黑） |
| 品牌点缀 | `#a2375f`（深玫红） |
| 背景 | `#fbf9f8`（暖白） |
| 卡片背景 | `#ffffff` |
| 正文 | `#4c4546`（深灰） |
| 字体 | Inter（正文）+ Hanken Grotesk（标题/数字） |

## 页面结构

1. **Hero 区** — 项目名称 + 一句话定位 + UCD 标签
2. **背景与痛点** — 四角色痛点 + 传统 vs 数字化对比
3. **UCD 设计流程总览** — 双钻模型可视化
4. **模块 1：多角色登录体系** — "角色即视图"设计
5. **模块 2：商品目录与 MOQ** — B端 vs C端差异 + Poka-yoke 防错
6. **模块 3：购物车与结算** — 授信扣款 + 合同式支付弹窗
7. **模块 4：供应端工作台** — 三个独立工作台
8. **组件库设计规范** — Design Tokens + 6 类组件
9. **页尾**

## 核心决策日志

| 决策 | 选择 |
|------|------|
| 项目网页类型 | 全新项目官网（非增量修改） |
| 目标受众 | 招聘/作品集 |
| 视觉风格 | 专业 B2B 产品感 |
| 技术形态 | 单页 HTML |
| 内容深度 | 4 精选模块 + 其余概要 |
| 页面架构 | 纵向叙事长卷 |
| 组件库位置 | 内嵌在单页末尾 |
